2.2 频数分布

2.2.1 频数分布表

例 从 adsl 数据集提取女性的基线 BMI (\(kg/m^2\)) 数据，结果如下：

adsl <- read_sas("data-adam/adsl.sas7bdat")
tb_bmi_bl <- adsl |> 
  select(BMIBL, SEX) |> 
  filter(SEX == "F") |> 
  select(BMIBL)

nrow(tb_bmi_bl)
#> [1] 143

共有143例数据，检查是否有缺失值：

tb_bmi_bl |> 
  anyNA()
#> [1] TRUE

输出结果为TRUE，提示有缺失值。利用下面的代码可以计算缺失值的个数：

tb_bmi_bl |> 
  is.na() |> 
  sum()
#> [1] 1

输出结果为1， 基线 BMI 中有1个缺失值。数据分析时可根据具体情况做相应处理，例如：

  1. 直接删除
tb_bmi_bl_clean <- na.omit(tb_bmi_bl)
nrow(tb_bmi_bl_clean) 
#> [1] 142
  1. 填补缺失值（简单起见，这里直接用均数）
tb_bmi_bl_filled <- tb_bmi_bl |> 
  mutate(
    BMIBL = ifelse(is.na(BMIBL), mean(BMIBL, na.rm = T), BMIBL)
  ) |> 
  round(digits = 1)

填补完成后，对143名女性的基线BMI 求 最小值、最大值和极差：

bmi <- tb_bmi_bl_filled
range(bmi)
range_bmi <- range(bmi) |>                               
  diff() |>
  print()
#> [1] 15.1 40.1
#> [1] 25
1
range()返回包含最小值和最大值的向量
2
利用管道操作符和diff()计算极差

也可用下面的方法计算：

min_bmi <- min(bmi)
max_bmi <- max(bmi)
range_bmi <- max_bmi - min_bmi

组段数通常取10~15组，这里取12；利用 seq() 函数得到组段的上下限：

breaks <- seq(from = min_bmi, to = max_bmi, length.out = 12)
breaks
#>  [1] 15.10000 17.37273 19.64545 21.91818 24.19091 26.46364 28.73636 31.00909
#>  [9] 33.28182 35.55455 37.82727 40.10000

使用cut()函数将数据分到指定的组段，并根据它们落在哪个区间对变量值进行编码；

value_bmi <- pull(bmi)
cut(value_bmi, breaks = breaks, include.lowest = T, right = F) |> 
  str_c() |> 
  c(rep(" ", ceiling(nrow(bmi) / 8) * 8 - nrow(bmi))) |>

2.4 离散趋势的描述

以例2-1某医院138名正常成年女性的红细胞数为例，计算描述离散趋势的统计指标：

2.4.1 极差

range(bmi) |> 
  diff()
#> [1] 25

2.4.2 四分位数间距

IQR(value_bmi) |> 
  round(digits = 2)
#> [1] 5.15

2.4.3 方差和标准差

var(value_bmi)
#> [1] 18.72291
sd(value_bmi)
#> [1] 4.326998

2.4.4 变异系数

cv <- (sd(value_bmi) / mean(value_bmi)) * 100
round(cv, digits = 2)
#> [1] 18.03

2.5 医学参考值范围

以例2-1某医院138名正常成年女性的红细胞数为例，确定其医学参考值范围：

med_ref_range <- quantile(value_bmi, probs = c(0.025, 0.975)) |> 
  round(digits = 3) |> 
  print()
#>  2.5% 97.5% 
#> 17.81 33.09

由上可知，正常成年女性红细胞数的95%医学参考值范围为: (17.81, 33.09)
